Добро пожаловать на агрегатор курсов валют - blackmarket.in.ua

Какая основная цель этого сайта?

Это предоставление информации из открытых источников о заявках физлиц на покупку и продажу валюты. С помощью графиков которые генерируются в режиме реального времени вы можете быстро и точно сориентироваться в ситуации с курсом доллара и перейти к онлайн сделке которая покажется вам интересной на том или ином сайте.

По каким валютам доступны данные?

На данный момент графики доступны только для пары доллар-гривна.

По каким регионам доступны данные?

На данный момент графики доступны только для Киева.

Как читать графики?

Графики имеют вид диаграммы распределения. Это сделано для того чтоб отразить распределения количества заявок по времени.
